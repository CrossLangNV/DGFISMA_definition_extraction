user_scripts.classifier_train module¶

user_scripts.classifier_train.main(train_sentences: ‘Location of training sentences’, train_labels: ‘Location of training labels’, model_storage_directory: ‘Location of output model directory’, validation_split: ‘Fraction of training data to use as validation data’, ‘option’ = 0.2, validation_sentences: ‘Location of validation sentences’, ‘option’ = None, validation_labels: ‘Location of validation labels’, ‘option’ = None, epochs: ‘number of epochs to train’, ‘option’ = 10)¶

Train the model

Parameters

  • train_sentences –

    path to file that contains input sentences example file:

    ”This is sentence 1

    While this is another sentence

    ”

  • train_labels –

    path to file that contains the labels (all integers) example file:

    ”1

    0

    ”

  • model_storage_directory – path to where model should be saved

  • validation_split – fraction of training data to be used as validation data. If validation_sentences and validation_labels are both not None, this will be ignored and whole training dataset is used.

  • validation_sentences – path to file with validation dataset sentences. Same format as train_sentences.

  • validation_labels – path to file with validation dataset labels. Same format as train_labels.

  • epochs – number of epochs to train the model.

Returns

trained model is saved in model_dir
